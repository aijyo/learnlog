上面是字符集，这个简单ocr是识别简单模式的 图片，假设图片上的所有字符都有严格的间隔不存在1_1这种连接不好切分的情况
简单办法就是：输出文字到图像上的时候跟上面一样，每个字符都间隔一个空格，大大降低了图片分割的难度（行之间都是可以简单分割的）
