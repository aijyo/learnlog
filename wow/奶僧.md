# 治疗循环场景（祥和宗师织雾武僧）

---

## A. 建立治疗节奏（开场/平稳期）
目标：快速铺满复苏之雾并保持高效治疗覆盖。

1. **预铺**：
   - 复苏之雾 → 复苏之雾（第二个目标）
   - 若有空档，释放禅意波或真气爆裂（双治疗+伤害）
2. **持续覆盖**：
   - 旭日东升踢 延长现有复苏之雾持续时间
   - Soothing Mist 连接主目标 → 瞬发真气波 / 活血术
3. **法力管理**：
   - 法力茶在大规模治疗前使用
4. **站位**：
   - 保持中近距离，方便治疗箭矢覆盖更多人

---

## B. 单体高额伤害应对（Boss猛击/高压伤害）
目标：保障关键队友（坦克或机制目标）存活。

1. **预防性覆盖**：
   - 生命之茧（提前 1~2 秒） → Soothing Mist 持续
2. **爆发治疗**：
   - 真气波（瞬发）+ 活血术 连续释放
   - 复苏之雾 保证 HoT 不断
3. **持续维护**：
   - Enveloping Mist + Renewing Mist 叠加
4. **应急**：
   - 自身受威胁时，躯不坏防御

---

## C. AOE 群体治疗（Raid/M+ 大群伤）
目标：高效群疗并覆盖全队 HoT。

1. **技能爆发顺序**：
   - 氤氲之雾（Essence Font） → 活血术（Vivify）x2~3 → 气吞山河（Revival）
2. **中期维持**：
   - 禅意波 / 真气爆裂 持续填充
   - Renewing Mist 铺满全团
3. **法力管理**：
   - Mana Tea + Essence Font 连动，节省蓝耗
4. **移动治疗**：
   - Chi Burst / Chi Wave / Renewing Mist 保证治疗不中断

# 祥和宗师（Mistweaver Monk）技能加成关系图

## 1. HOT 延长相关
**回天（Rising Mist）**
  → 旭日东升踢（Rising Sun Kick）【命中后】
    → 复苏之雾（Renewing Mist）（持续时间 +4s，最多至初始的 100%）
    → 真气波（Essence Font）（持续时间 +4s，最多至初始的 100%）
    → 氤氲之雾（Enveloping Mist）（持续时间 +4s，最多至初始的 100%）

## 2. 复苏之雾相关
**振奋之雾（Mist Wrap）**
  → 复苏之雾（Renewing Mist）（治疗量 +10%，持续时间 +2s）

**振魂引（Uplifted Spirits）**
  → 活血术（Vivify）【命中带有复苏之雾的目标时】
    → 额外治疗所有受你复苏之雾影响的目标

**生生不息（Chi-Ji's Blessing / Misty Peaks）**
  → 复苏之雾跳跃目标数 +1，并在跳跃时触发额外效果（视具体天赋版本）

## 3. 抚慰之雾相关
**氤氲之雾（Enveloping Mist）**
  → 抚慰之雾（Soothing Mist）引导中可瞬发施放
  → 提高目标受治疗效果 +40%

**迅如影（Soothing Voice / Lifecycles）**
  → 抚慰之雾引导后施放活血术（Vivify）或氤氲之雾（Enveloping Mist）消耗降低 / 提高治疗效率

## 4. 伤害转治疗（拳法机核心）
**祥和宗师（Ancient Teachings）**
  → 真气爆裂（Essence Font）/ 雷光茶（Thunder Focus Tea）激活后
    → 你的近战技能（如旭日东升踢、猛虎掌、神鹤引项踢）将对附近受伤的盟友造成治疗（持续 15s）

**回天（Rising Mist）**
  → 搭配旭日东升踢循环保持 HOT，配合祥和宗师提升总治疗量

## 5. 雷光茶（Thunder Focus Tea）相关
**雷光茶（Thunder Focus Tea）**
  → 下一次活血术（Vivify）无消耗
  → 或下一次复苏之雾（Renewing Mist）无冷却
  → 或真气波（Essence Font）施放时间减半

## 6. 群疗覆盖加成
**真气波（Essence Font）**
  → 赋予目标“真气灌注”buff（额外受到你的直接治疗时获得额外治疗）
  → HOT 可被回天延长

**活血术（Vivify）**
  → 治疗主目标，并对所有带复苏之雾的目标额外治疗一次
